# Stories bank

Behavioural interview answers prepared in advance. Each story follows the STAR format:
**Situation → Task → Action → Result**

Keep each story to ~2 minutes spoken. Revise after each interview based on what landed.

---

## How to use this bank

Before each interview:
1. Review the job description for likely behavioural themes
2. Select 4–6 stories that match
3. Read them aloud — they should sound natural, not recited
4. After the interview, note which stories you used and how they landed
